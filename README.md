# Snake Game🐍

## Summary

This ia a simple CLI based game insipired the local snake game 😉 on phones.


## Usage🛠️ 


1. **Clone the Repository**
   ```sh
   git clone https://github.com/Awungia112/snake_game.git
   cd fibbot
   ```
2. **Build and run the Project**
   ```sh
   cargo run
   ```
